# ElectroExpress - интернет-магазин электроники

## Общее описание

ElectroExpress - это функциональный макет веб-сайта магазина электроники с интуитивно понятным интерфейсом, системой управления товарами и оформлением заказов. Проект разработан с использованием современных технологий и архитектурных подходов, что делает его надежным и легко расширяемым решением для электронной коммерции.

## Функционал

### Для пользователей:

- Просмотр каталога товаров с возможностью фильтрации
- Добавление товаров в корзину
- Оформление заказов
- Персональный кабинет
- Просмотр информации о товарах (описание, цена, скидки)

### Для администраторов:

- Управление товарами (добавление, редактирование, удаление)
- Управление скидками на товары

## Технологический стек

### Фронтенд:

- React.js
- TypeScript
- Vite
- React Router

### Бэкенд:

- Node.js с Express.js
- MySQL
- RESTful API

## База данных

Проект использует MySQL для хранения данных. База данных содержит следующие основные таблицы:

- `products` - товары
- `users` - пользователи
- `cart_items` - товары в корзинах
- `orders` - заказы
- `order_items` - товары в заказах

Скрипт для автоматической настройки базы данных включен в проект и запускается при первом старте.

## Как запустить проект

### Предварительные требования:

- Node.js
- MySQL
- Git

### Шаги для запуска:

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/shetkir/electroexpress-web.git
   cd electroexpress-web
   ```
2. Установите зависимости:

   ```bash
   cd htdocs
   npm install
   ```
3. Настройте параметры подключения к базе данных в файлах `db-setup.js` и `simple-server.js` (при необходимости)
4. Запустите проект:

   ```bash
   ./start.sh
   ```
5. Приложение автоматически откроется в вашем браузере

## Возможности для расширения

Данный проект имеет модульную структуру, что позволяет легко расширять его функционал:

1. **Добавление платежной системы** - интеграция с платежными шлюзами для онлайн-оплаты
2. **Система категорий товаров** - детальная фильтрация по категориям
3. **Отзывы и рейтинги** - возможность оставлять отзывы о товарах
4. **Аналитика продаж** - панель для администратора с отчетами
5. **Мобильное приложение** - создание мобильной версии на базе React Native
6. **Мультиязычность** - поддержка нескольких языков интерфейса
7. **Система рекомендаций** - алгоритмы для персонализированных предложений
